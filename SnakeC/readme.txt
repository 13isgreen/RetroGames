This is a retro game written in C,
designed to run on most (Windows,MacOS,Linux) with minimum modifications.
The game relies son the SDL2 (Simple Direct Media Layer)

These are the recocmmended installation prccess from their documentation

Windows

    Download SDL2 from: SDL2 Downloads
    Extract it and place the include and lib folders somewhere accessible.
    Link against SDL2.lib and SDL2main.lib when compiling.

Linux (Ubuntu/Debian)

sudo apt update
sudo apt install libsdl2-dev

macOS

brew install sdl2
